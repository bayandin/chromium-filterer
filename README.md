# chromium-filterer

Extracts [`chrome/test/chromedriver`](https://chromium.googlesource.com/chromium/src/+/main/chrome/test/chromedriver) from the [Chromium](https://github.com/chromium/chromium) monorepo with full git history and publishes it to [bayandin/chromedriver](https://github.com/bayandin/chromedriver).
